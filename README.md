# SSS Contribution Penalty Calculator

A professional, web-based utility designed to help members and employers estimate penalty fees for late Social Security System (SSS) contributions. This project features a clean, mobile-responsive interface that aligns with the SSS corporate branding.

---

## 🚀 Features

* **Penalty Estimation:** Navigate to the calculator tool to compute late payment interests.
* **Official Integration:** Direct links to the official SSS website for the latest updates and contribution tables.
* **Modern Design:** A responsive "Card UI" built with clean CSS, ensuring usability on desktops, tablets, and smartphones.
* **Quick Support:** Easy access to branch-specific contact information.
* **Holidays:** Holiday dates are from 2018 up to present based on https://www.officialgazette.gov.ph/nationwide-holidays/

---

## 🛠️ Technology Stack

* **HTML5:** Semantic structure for the web interface.
* **CSS3:** Custom styling featuring Flexbox for layout and smooth hover transitions.
* **SVG Support:** High-quality official branding icons.

---

## 📂 Project Files

| File | Description |
| :--- | :--- |
| `index.html` | The main landing page and navigation hub. |
| `contribution_calculator.html` | The core tool for calculating penalty amounts. |

---

## 🖥️ Getting Started

1.  **Download** the repository files to your local folder.
2.  Ensure both `index.html` and `contribution_calculator.html` are in the same directory.
3.  **Open** `index.html` in any modern web browser (Chrome, Edge, Firefox, or Safari).
